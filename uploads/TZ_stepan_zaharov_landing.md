# ТЗ: Личный сайт-визитка Степана Захарова (кредитный брокер)

## 0. Контекст и роль

Ты разрабатываешь личный сайт-визитку для Степана Захарова — кредитного брокера из Санкт-Петербурга. Это НЕ лендинг с лидформой, а имиджевый сайт, цель которого — конвертировать посетителя в подписку/обращение в Telegram через демонстрацию экспертности и проверяемых результатов.

**Дизайн-система уже загружена в проект** (цвета, шрифты, токены, компоненты) — используй её для всех элементов. Референс по духу (не копировать буквально, только настроение): https://georgiirivera.ai/ — светлая, современная, с акцентом на личность через крупную типографику и сдержанную анимацию при скролле. Прояви креативность в композиции, но визуальный язык (цвета/шрифты/радиусы) — строго из дизайн-системы.

**Логотип:** замени логотип дизайн-системы на логотип, который придёт отдельным файлом вместе с этим ТЗ.

**Источник данных:** вместе с этим ТЗ прикреплён файл `messages.html` (экспорт Telegram-канала «Кредитное Королевство», @TheCreditKingdom) со всеми фото и видео в подпапках `photos/` и `video_files/`. Все факты, кейсы и цифры ниже взяты из этого канала — используй сам файл и медиафайлы как первоисточник для верстки галерей и блоков кейсов.
