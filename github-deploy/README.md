# Сайт «Степан Захаров — Кредитное Королевство»

Одностраничный сайт кредитного брокера. Полностью самодостаточный: весь код,
стили, скрипты и изображения зашиты внутрь `index.html` — соседних файлов не требуется.

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub (например, `step-site`).
2. Загрузите в него содержимое этой папки: `index.html`, `.nojekyll`, `.gitignore`.
3. Settings → Pages → Source: ветка `main`, папка `/ (root)` → Save.
4. Через ~1 минуту сайт откроется по адресу `https://<логин>.github.io/<репозиторий>/`.

## Что подгружается из интернета

- Шрифт Manrope (Google Fonts); без сети — системный фолбэк.
- Видео Rutube в блоке «Выступления».
- Переходы по кнопкам Telegram и Max.

Текст, фото и скрины работают офлайн — они внутри файла.

## Проверить перед публикацией

- Кнопки «Написать в Макс» ведут на заглушку `https://max.ru/` — заменить на реальную ссылку.
- Telegram-ники в блоке «Бесплатная консультация»: `@FinansistStep`, `@StepCredit`, `@TheCreditKingdom`.
